# node_bolg

> nodejs的自学实践项目，前端使用页面angular搭建，后台管理使用ejs模板，后台技术栈 nodejs + mySQL ,初步设想为搭建一个博客web，

# 文件结构
>
|--libs                    自定义的库
     |--common.js          MD5签名
|--node_modules            依赖文件
|--static                  静态文件
     |--upload             用户上传的文件
|--template                模板
     |--admin
          |--components    公用组件文件夹
          |--
     |--web
|--route                   路由
     |--admin
          |--login.ejs     管理员登录 
          |--index.ejs     后台管理首页
          |--banners.ejs   后台管理banner页
          |--custom.ejs    后台管理custom页
     |--web
|--package.json
|--server.js
